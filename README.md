Nmap LAN Reconnaissance Lab

Overview

This project demonstrates a complete network reconnaissance workflow using Nmap in a controlled LAN environment. The labs cover host discovery, port scanning, service version detection, NSE scripting, SSL/TLS analysis, and OS fingerprinting against a Huawei home router.

The purpose of this project is to understand how penetration testers and security researchers perform reconnaissance and analyze exposed network services using Nmap.

---

Objectives

- Discover live hosts in a LAN
- Identify exposed ports and services
- Perform service fingerprinting
- Use NSE scripts for enumeration
- Analyze SSL/TLS configurations
- Perform OS detection and traceroute
- Document a professional reconnaissance workflow
